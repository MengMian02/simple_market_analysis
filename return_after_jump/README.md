# return_after_jump

This mini-research seeks to understand the behaviour of the security on the days after a shock.

Behaviour can be defined as (mean return, median return, percentage of positive returns, volume)

Shock can be defined as (2*rolling standard deviation etc.)

While this area has already been well-studied, this mini-research aims to use simple and introductory tools to understand
this problem. The main purpose is to acquire new skills along the way
